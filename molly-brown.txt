            Make changes to the coding project.
            Commit these changes often (this is the version control). 
                git add 
                git commit -m 'a message'
                git push origin -u molly-brown
            When you are ready to push your changes to your branch: complete these git commands -->
                git add 
                git commit -m 'a message'
                git push
            Create a Pull-Request to submit my work: using my own student fork as the base.
